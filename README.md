# 垃圾图像分类CNN模型
基于卷积神经网络实现垃圾图片分类（纸板、玻璃、金属、纸张、塑料、其他垃圾）。

## 项目介绍
本项目使用 PyTorch 搭建简单 CNN 网络，对垃圾图像进行 6 分类。
包含**数据加载、模型训练、准确率测试、结果绘图**功能。
